# 📱 BCA Android Practical Slips & Project

## 📌 Overview

This repository contains solutions to **previous year BCA Android practical slips** along with an Android project developed as part of academic learning.

The aim of this repository is to help students understand Android concepts and prepare for practical exams.

---

## 📂 Contents

### 🧾 Practical Slips

* Solutions to last year BCA Android practical slips
* Covers basic to intermediate Android concepts
* Includes UI design, activities, intents, and more

### 🚀 Android Project

* Complete Android application developed for academic purpose
* Demonstrates real-world implementation of Android concepts

## ▶️ How to Run the Project

1. Install **Android Studio**
2. Clone this repository:

   ```bash
   git clone https://github.com/Piyush18aug/Android_Project_and_slips_BCA.git
   ```
3. Open the project in Android Studio
4. Let Gradle sync complete
5. Run the app on Emulator or Physical Device
